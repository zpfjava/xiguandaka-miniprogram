# 云数据库集合创建指南

## 必须手动创建（微信云开发限制）

云数据库集合无法通过云函数代码自动创建，必须在**云开发控制台**手动操作。

## 操作步骤

1. 打开**微信开发者工具**
2. 点击工具栏 **「云开发」** 按钮 → 进入云开发控制台
3. 左侧菜单选择 **「数据库」**
4. 点击右上角的 **「+」** （添加集合）按钮
5. 依次输入以下 11 个集合名称，每输入一个点确认：

### 需要创建的集合（按顺序）

| 序号 | 集合名称 | 说明 | 权限建议 |
|------|---------|------|---------|
| 1 | `users` | 用户表 | 所有用户可读，仅创建者可写 |
| 2 | `study_plans` | 学习计划表 | 所有用户可读，仅创建者可写 |
| 3 | `checkins` | 打卡记录表 | 所有用户可读，仅创建者可写 |
| 4 | `wishlists` | 愿望清单表 | 所有用户可读，仅创建者可写 |
| 5 | `points_history` | 积分历史表 | 所有用户可读，仅管理云函数可写 |
| 6 | `daily_checkins` | 每日签到表 | 所有用户可读，仅创建者可写 |
| 7 | `user_achievements` | 用户成就表 | 所有用户可读，仅管理云函数可写 |
| 8 | `achievements` | 成就定义表 | 所有用户可读，仅管理云函数可写 |
| 9 | `feedbacks` | 用户反馈表 | 所有用户可读，仅创建者可写 |
| 10 | `parent_binds` | 家长绑定表 | 所有用户可读，仅创建者可写 |
| 11 | `sms_codes` | 短信验证码表 | 仅管理云函数可读写 |

## 创建完成后的权限设置

每个集合创建后，点击集合名 → **「权限设置」** → 推荐选择：

- `users`, `study_plans`, `checkins`, `wishlists`, `daily_checkins`, `feedbacks`, `parent_binds`
  → 选择 **「仅创建者可读写」** 或 **「所有用户可读，仅创建者可写」**

- `points_history`, `user_achievements`, `achievements`, `sms_codes`
  → 选择 **「仅管理端可读写」**（这些只需要云函数操作）

## 初始化成就数据

所有集合创建完成后：
1. 进入云开发控制台 → 云函数 → 找到 `dbInit`
2. 点击「测试」→「运行测试」（此时 achievements 集合已存在，会初始化10条预设成就数据）
3. 或者直接在 `achievements` 集合中手动添加记录

## 验证

创建完成后，在数据库页面应该能看到 11 个集合。
然后可以在小程序中测试登录功能。
